# Pricing Placeholders for Aurora Rayes Pet Care Agent

Payments are disabled by default. These prices are planning placeholders, not live offers.

| Plan | Placeholder price | Best for |
| --- | ---: | --- |
| Free | $0 | Trial users and demos |
| Pro Monthly | $7/month | Individuals and small teams |
| Pro Annual | $59/year | Committed users |
| Partner | Custom | White-label, integrations, and service packages |

Before enabling payments:
1. Replace placeholder environment variables with live provider values outside source control.
2. Set the payment enablement flag to `true` only in the deployment environment.
3. Publish final support, refund, privacy, and terms pages.
4. Replace `REPLACE_WITH_CHECKOUT_LINK_WHEN_PAYMENTS_ENABLED` with the approved checkout URL.
