# Aurora Rayes Pet Care Agent

AI-assisted pet care planning, symptom triage, vaccination tracking, diet guidance, and breed insights for pet owners and care teams.

## Aurora Rayes ecosystem fit
Aurora Rayes Pet Care Agent is positioned as a focused Aurora Rayes product for pet parents, veterinary support teams, shelters, breeders, and pet-care brands. It can be sold standalone, bundled with related Aurora Rayes agents, or packaged as a white-label starter for operators who need a polished AI workflow quickly.

## Core features
- Symptom checker
- Vet Q&A intake
- Vaccination tracker
- Diet planner
- Breed insights

## Commercial use cases
- Launch a niche AI assistant under the Aurora Rayes umbrella.
- Use as a lead magnet or paid workflow for pet parents, veterinary support teams, shelters, breeders, and pet-care brands.
- Bundle with onboarding, content, or consulting services.
- Extend with customer-specific integrations after discovery.

## Monetization status
Pricing and checkout copy are placeholders. No live payment links or live Stripe keys are committed. Payments are disabled by default and must be enabled only after valid environment variables, terms, and support processes are configured.


## Setup
```bash
npm install
npm run dev
```

Copy `.env.example` to `.env.local` and add provider credentials when ready. `PAYMENTS_ENABLED=false` is the safe default.

## Build and validation
```bash
npm run build
npm run lint
```
